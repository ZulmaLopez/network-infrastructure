\# network-infrastructure



Diseño e implementación de red empresarial con VLANs, enrutamiento inter-VLAN, 

HSRP y telefonía IP Cisco CME usando Cisco Packet Tracer.



\## Tech Stack



!\[Cisco](https://img.shields.io/badge/Cisco-1BA0D7?style=for-the-badge\&logo=cisco\&logoColor=white)

!\[Packet Tracer](https://img.shields.io/badge/Packet\_Tracer-1BA0D7?style=for-the-badge\&logo=cisco\&logoColor=white)



\## Topología de Red



| Componente | Detalle |

|---|---|

| VLANs | Segmentación por departamentos |

| Inter-VLAN Routing | Router-on-a-stick |

| HSRP | Alta disponibilidad con gateway redundante |

| Telefonía IP | Cisco CME con teléfonos IP |

| Servicios | DHCP, DNS, HTTP en servidor dedicado |



\## Configuraciones implementadas



\- VLANs configuradas en switches Cisco

\- Enrutamiento inter-VLAN

\- Protocolo HSRP para redundancia de gateway

\- Central telefónica Cisco CME

\- Servicios de red: DHCP, DNS, HTTP
